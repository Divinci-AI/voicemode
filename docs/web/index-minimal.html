Voice Mode

Voice conversations for AI assistants.

Voice Mode enables natural voice interactions with Claude, ChatGPT, and other AI assistants. Built on the Model Context Protocol (MCP) for reliability and simplicity.

Watch Demo →
# Install pip install voice-mcp npm install voicemode # Use with Claude claude mcp add voice-mcp uvx voice-mcp

Features

  • Natural voice conversations with AI
  • Works with OpenAI API and compatible services
  • Local or cloud speech processing
  • LiveKit integration for production use
  • Simple setup, powerful capabilities

Requirements

  • OpenAI API key (or self-hosted alternative)
  • Microphone and speakers
  • Python 3.8+

Why Voice Mode?

We built Voice Mode to make voice AI accessible. No complex setup, no vendor lock-in, just clean code that works. Whether you're building a personal assistant or deploying to thousands of users, Voice Mode scales with you.
